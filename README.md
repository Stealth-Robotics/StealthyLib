# StealthyLib
This library will be a repository for all the common code that Stealth tends to use throughout each FRC season. Team projects can use this library to keep development quick and efficient.

## Installation Instructions

### Pre-installation
There are a couple other vendor libraries that Stealthy Lib relies on in order to function. They are listed below along with their JSON URL to install them with WPILib's `Manage Vendor Libraries -> Install New Libraries (Online)` tool.

#### Phoenix (CTRE)
Currently the version used by the Stealthy Lib is the 2022 version. Here is the link to install the vendor library:
```
https://maven.ctr-electronics.com/release/com/ctre/phoenix/Phoenix-frc2022-latest.json
```

#### REV
Currently StealthyLib is using the 2021 version of the library. The link to the vendor library:
```
https://software-metadata.revrobotics.com/REVLib.json
```

#### Pathplanner
Currently Stealthy Lib is using the 2023 Beta version of the library. Link:
```
https://3015rangerrobotics.github.io/pathplannerlib/PathplannerLib.json
```

### Stealthy Lib Installation
Similarly to the libraries in the pre-installation instructions, Stealthy Lib has also been created as a vendor lib, simply use the link below with WPILib's `Manage Vendor Libraries -> Install New Libraries (Online)` tool, exactly like the libraries before.
